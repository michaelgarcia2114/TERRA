# TERRA

**TERRA** is a NASA Space Grant research rover designed to collect and analyze environmental data to support Earth science applications. The project focuses on autonomous field monitoring by combining robotics, embedded systems, and sensor fusion to measure soil and environmental conditions in real time.

Built on a Raspberry Pi and Arduino architecture, TERRA integrates multiple sensors, motor control, and LiDAR-based navigation to gather accurate field data while operating autonomously. The collected information can be used to support research in areas such as drought monitoring, precision agriculture, environmental mapping, and natural resource management.

## Project Goals

* Develop an autonomous ground rover for environmental data collection.
* Measure soil moisture and other environmental conditions using onboard sensors.
* Navigate and map terrain using LiDAR and onboard computing.
* Integrate embedded hardware with robotics software for reliable field operation.
* Provide a scalable platform for future AI and computer vision research.

## Current Status

TERRA is currently under active development as part of the NASA Space Grant Program. Hardware integration, autonomous navigation, sensor validation, and software development are being completed iteratively, with new features and documentation added throughout the project.
